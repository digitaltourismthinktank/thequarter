Icon-only control — square, soft corners, 44px default hit target. Always pass `label` for accessibility.

```jsx
<IconButton icon="bell" label="Notifications" />
<IconButton icon="menu" label="Open menu" variant="ghost" />
<IconButton icon="arrow-right" label="Next" variant="solid" />
```
Variants: `soft` (sand fill, default), `outline`, `ghost`, `solid` (ink).
