Quiet status pill. Room availability uses `available` / `busy` / `soon`; `gold` for premium markers; `ink` for high-contrast.

```jsx
<Badge tone="available" dot>Available now</Badge>
<Badge tone="busy" dot>In use</Badge>
<Badge tone="gold" icon="check">Included</Badge>
<Badge tone="neutral" size="sm">8–10 people</Badge>
```
