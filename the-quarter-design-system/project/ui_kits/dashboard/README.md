# Dashboard UI kit — The Quarter

The member dashboard and web app — designed to feel like a calm, premium product, not an admin panel.

**Open:** `index.html`.

**Views** (left-sidebar nav, `Dashboard.jsx`)
- **Overview** — greeting, plan & days-remaining stat tiles, upcoming booking, an inline week's availability to book against, the **Quarter Card** + "Add to Apple Wallet", and a perk for you.
- **Book a room** — room switcher + full weekly `AvailabilityCalendar` with a sticky booking rail (package, days off plan, reserve).
- **Perks** — the full partner-perk catalogue with live redeem.
- **Account** — details, notification switches, plan upgrade nudge.

Composed entirely from bundled components (`StatTile`, `QuarterCard`, `AvailabilityCalendar`, `PerkCard`, `RoomCard`, forms, `Avatar`, `Badge`, `Button`). Sign-out links back to the website kit's login; the website login links here.
