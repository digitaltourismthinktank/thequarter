# Display UI kit — The Quarter

A wall-mounted, glanceable, landscape board for the office — brand expression as much as utility.

**Open:** `index.html` (auto-scales a 1920×1080 board to any screen; designed for a landscape display).

Shows, readable across the room:
- **Header** — welcome, the day's date, a live clock, location.
- **Meeting rooms** — three large cards with big status chips (Available / In use), capacity and the next free time. High-contrast warm palette, **no green** — available reads gold, in-use reads terracotta.
- **What's on today** — the day's socials, talks and workshops.

Large type, strong contrast and the brand's ink-and-gold language for legibility from a distance.
