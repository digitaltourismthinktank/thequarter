# App UI kit — The Quarter

Phone-scale member app, showing the two highest-value actions to prove the system stretches to mobile.

**Open:** `index.html` (two phones side by side).

- **Book a room** — room header with live status, a horizontal day selector, tappable time-slot list (free / held / booked), and a sticky reserve action.
- **Perks & Quarter Card** — the Quarter Card at phone scale (QR hidden for the compact view) above a redeemable perk list, with a bottom tab bar.

Built from the same bundled components (`QuarterCard`, `Badge`, `Button`, `Avatar`, `Icon`) inside a lightweight phone shell. Indicative screens — not the full app.
